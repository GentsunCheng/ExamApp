# 学习进度

学习进度用于将学习内容拆分为“模块/任务”，并记录每个用户的任务状态。

## 数据存储

- 进度数据库：`~/.exam_system/progress.db`

## 状态与颜色

- `未开始`：灰色
- `进行中`：主题主色
- `已完成`：绿色

## 管理员端使用

- 入口：管理员面板 → `学习进度`
- 常用操作：
  - `导出模板`：生成 Excel 模板，按模板填写模块与任务
  - `导入模板`：从 Excel 批量导入模块/任务
  - `覆盖导入`：勾选后导入会清空“同名模块”下的任务与用户进度记录再重建
  - `导出用户进度`：导出指定用户的当前进度（包含状态、更新时间、更新人）
  - 状态修改：在用户任务列表的“状态”列下拉选择（会即时写入 `progress.db`）

## 普通用户端使用（只读）

- 入口：用户中心 → `学习进度`
- 特性：
  - 模块分组展示任务列表与状态
  - 仅展示，不提供编辑入口，也不支持快捷键修改
  - 在“考试/历史/学习进度”之间切换不会改变任何进度数据

## Excel 模板格式

- 工作表：
  - `说明`：模板说明（导入时会跳过）
  - 其他工作表：每个工作表代表一个“模块”，工作表名即模块名
- 表头（第 1 行）：
  - `任务名`：必填；模块内唯一
  - `描述`：可选
  - `顺序`：可选；整数；用于排序
- 注意：
  - 会跳过工作表名为 `说明` / `_meta` / `meta` 的表
  - 空的“任务名”行会被忽略
